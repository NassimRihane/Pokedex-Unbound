# Pokedex Unbound

This project is an iPhone app made to track Pokemon through the main games.


## Overview

It features the following functionalities for all Pokemon:
- Display all Pokemon available from Generation 1 to 8, with multiple layouts
- Search bar returning name in English, Japanese romaji or French
- Sort them according to a chosen stat
- Select only Pokemon from specific generations or type combination
- Filtering through minimum statistics
- Filtering through capture status
- A comparison page, to check the statistics of 2 given Pokemon

And on individual pages:
- Display the Pokemon information and statistics
- Access to moves learned through experience and HM/TM
- Informations on Pokemon location
- A capture tracker for specific vesions

## Installation

To install the individual data for all Pokemon, first execute the Python file "data_searcher.py". Make sure to have a stable Internet connection.

Compile the Swift code and export it to a valid target (preferably an iPhone) to use the app. I recommend using Xcode
